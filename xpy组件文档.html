1.JD轮播组件

简介

  • 当有一定数量的图片需要展示时，需要对图片进行排序，依次展示，可随意点击查看第几张图片时，可适用该组件

配置项

参数名称 说明 类型 默认值 可选值 是否必写
data 轮播图片数据地址 array - - 是
isInfinit 是否循环轮播 Boolean true true/false 否
isSlide 轮播效果是否是滑动, false效果是闪烁 Boolean flase true/false 否
isBtn 轮播图是否有左右按钮 Boolean true true/false 否
skipBar 轮播圆点 object - true/false click/hover 否
slideTime 每张图片轮播的时间 number - - 否

props

字段名 说明 是否可选项 类型 默认值
show 是否有轮播的圆点 必选 Boolean true
trigger 轮播圆点的触发方式 必选 - hover

基本用法

html代码

<div class="lunbo"></div>

javascript代码

var data1 = [{
        'url': './images/1.jpg'
    },{
        'url': './images/2.jpg'
    },{
        'url': './images/3.jpg'
    },{
        'url': './images/4.jpg'
    },{
        'url': './images/5.jpg'
    }];
$('.lunbo').carousel({
    'data': data1,
    'isSkip': true,
    'isInfinit': true,
    'isSlide': true,
    'skipBar': {
        'show': true,
        'trigger': 'click'
    }
    'slideTime': 2000
})

2.JD滑动组件

简介

  • 当有一定数量的图片需要展示，手动点击滑动展示，可适用该组件

配置项

参数名称 说明 类型 默认值 可选值 是否必写
data 轮播图片数据地址 array - - 是
isInfinit 是否循环轮播 Boolean true true/false 否
slideLength 每次滑动的图片张数 number 5 - 否
showLength 图片可展示的张数 number 5 - 否
showImgContentWidth 每张图片的宽度大小 number 200 - 否

基本用法

html代码

<div class="slide"></div>

javascript代码

var data1 = [{
        'url': './images/1.jpg'
    },{
        'url': './images/2.jpg'
    },{
        'url': './images/3.jpg'
    },{
        'url': './images/4.jpg'
    },{
        'url': './images/5.jpg'
    }];
$('.slide').slideMove({
    'data': data1,
    'isInfinit': true,
    'slideLength': 5,
    'showLength': 5,
    'showImgContentWidth': 200
})

<div class="search-area"></div>

    "data": [],

    "searchIcon": "",

5.内容弹窗组件

简介

  • 当点击一个元素，页面弹出一个弹窗，弹窗名字可自定义，弹窗内容也可自定义，弹窗可以自定义个数和名字，支持open、close方法。

配置项

参数名称 说明 类型 默认值 可选值 是否必写
title 弹窗名字 string 弹窗 - 否
content 弹窗里的内容 - null - 否
area 弹窗的大小 array auto - 否
btn 弹窗的按钮 array 确定 - 否
success 弹窗成功后的回调 function - - 否
btn1 弹窗第一个按钮操作 function - - 否
btn2 弹窗第二个按钮操作 function - - 否

基本用法

javascript代码

var dia1 = $.open({
    'title': '弹窗',
    'content': html,
    'area': ['400px', '400px'],
    'btn': ['确定','取消'],
    'success': function(){

    },
    'btn1': function(){
        alert('a');

    },
    'btn2': function(){
        $.close(dia1);//关闭弹窗
    }
})

6.下拉选择框组件

简介

  • 点击一个文本框，弹出一个下拉选择框，使用下拉菜单展示并选择内容。

配置项

参数名称 说明 类型 默认值 可选值 是否必写
data 下拉选择框各项数据内容 array [] - 否
placeholder 下拉触发的文本框内的初始化内容 string null - 否
width 下拉触发的文本框的宽度大小 number auto - 否

props

字段名 说明 是否可选项 类型 默认值
value 选项的值 必选 string -
text 选项展示的文本 必选 string -
comment 选项的备注 可选 string -
selected 选项是否选中 可选 boolean -

基本用法

html代码

<div class="drop-select"></div>

javascript代码

var listData = [{
    'value': 'aaa',
    'text': 'aaa',
    'comment': '我是备注',
    'selected': true
},{
    'value': 'bbb',
    'text': 'bbb',
    'comment': '我是备注'
},{
    'value': 'ccc',
    'text': 'ccc',
    'comment': '我是备注'
}];
$('.drop-select').dropdown({
    'data': listData,
    'placeholder': '请选择',
    'width': '100%'
})
